# 🤖 Bot de Grid Trading para POL/USDT

## 🎯 Objetivo
Aprender a programar un bot de grid trading para operar en el par POL/USDT.

## 📁 Archivos
- `grid_bot.py` - El código principal del bot
- `config.json` - Configuración del bot
- `README.md` - Este archivo

## 🚀 Cómo usar
1. Revisa el código en `grid_bot.py`
2. Modifica `config.json` si quieres cambiar parámetros
3. Ejecuta el bot (necesitas Python local o GitHub Codespaces)

## 📊 Explicación del código
- **GridTradingBot**: Clase principal
- **calcular_niveles_grid()**: Calcula precios de compra/venta
- **simular_operacion()**: Simula una operación
- **ejecutar()**: Bucle principal
